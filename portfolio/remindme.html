RemindMe - Case Study

RemindMe was conceived in a Mobile Application Marketing class. The main premise of the application is to keep track of common tasks that one might have to complete about any aspect of life and set reminders for it. The app had to be designed with the user in mind in terms of usability and experience.

Process

From a design standpoint, the team decided to go with something simple yet beautiful that would allow the user to quickly learn how to use the application. I was in charge of wireframing the application user interface. I used Sketch as my tool of choice, and created a wireframe and prototype with Invision to give a demo when we presented the project
